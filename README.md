## ğŸ§© Installation

Clone the repository:

```bash
git clone https://github.com/SrushtiV12/BUZZHUB-.git
cd BUZZHUB-
```

Install Backend Dependencies

```bash
cd backend
npm install
```
Install Frontend Dependencies
```bash
cd ../frontend
npm install
```

##Running the App
Start the Backend
```bash
cd backend
npm run dev
```
Start the Frontend
```bash
cd frontend
npm start
```
